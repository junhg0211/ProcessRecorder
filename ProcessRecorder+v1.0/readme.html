ProcessRecorder를 사용해 주셔서 감사합니다.

사용법

  1. ProgramRecorder 폴더를 C:\Program Files (x86)위치로 옮긴다.
  2. 실행-끗

선택사항

  • %AppData%\Microsoft\Windows\Start Menu\Programs에 ProcessRecorder.exe의 바로가기를 만든다.
    • 시작 화면에서 검색으로 찾아서 실행할 수 있음!
  • %AppData%\Microsoft\Windows\Start Menu\Programs\Startup에 ProcessRecorder.exe의 바로가기를 만든다.
    • 컴퓨터 켜자마자 프로그램이 실행되게 만들 수 있음!
